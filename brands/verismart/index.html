La plataforma

Qué es VeriSmart y cómo está construida

VeriSmart es la aplicación web central de Veritas.cl. Sirve tanto a clientes internos como externos para la gestión de procesos de certificación y auditoría. Todo el stack vive en contenedores Docker.

docker-compose.yml 
version: '3.9'
services:

  frontend:
    build: ./frontend  # Vue.js 3
    ports:
      - "3000:3000"
    environment:
      - VITE_API_URL=http://api:8080

  api:
    build: ./backend   # Node.js 20
    ports:
      - "8080:8080"
    depends_on:
      - mongo
    environment:
      - MONGO_URI=mongodb://mongo:27017/vs

  mongo:
    image: mongo:7
    volumes:
      - mongo_data:/data/db

volumes:
  mongo_data:
⬡

Frontend SPA

Vue.js 3 + Vite. Aplicación de página única con routing del lado del cliente. Build estático servido por Nginx dentro del contenedor.

⚡

API REST

Node.js 20 LTS + Express. Autenticación JWT, middleware de roles, integración con servicios de Veritas.cl.

🍃

Base de datos

MongoDB 7 con esquemas Mongoose. Documentos flexibles para estructuras de certificación variables y auditorías históricas.

El problema

AWS nos está costando caro en todos los sentidos

La configuración actual en AWS EC2 + RDS fue válida para lanzar rápido, pero hoy presenta problemas de costo, latencia y complejidad operacional que frenan la evolución del producto.

💸
Costo desproporcionado
Pagamos servicios enterprise de AWS (EC2 t3.large + RDS + ALB + NAT Gateway) para una app que no justifica esa complejidad ni ese gasto mensual.
🌎
Sin región en LatAm
La instancia está en us-east-1 (Virginia). Los usuarios en Chile sufren ~280ms de latencia en cada request. El estándar UX objetivo es bajo 100ms.
🐌
Deploy lento y manual
Cada deploy requiere 18 minutos entre push, build en CodeBuild y rollout en ECS. Iterar rápido se hace cuesta arriba.
🔧
Overhead operacional alto
Security groups, VPC, subnets, IAM roles... Configuración que consume tiempo de ingeniería que debería estar en el producto.
📈
Costos que solo suben
AWS ajusta precios y agrega cargos (data transfer, requests, storage). Sin un plan, el costo anual supera los $22.000 USD.
La solución

Comparativa de proveedores

Evaluamos 5 opciones considerando costo, latencia para Chile, compatibilidad con nuestro stack Docker y calidad del soporte.
